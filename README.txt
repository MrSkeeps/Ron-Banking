Ron-Banking - A simple banking mod for FiveM with a 
deposit, withdraw and transfer system

Links

Preview - https://streamable.com/qotc36

Forum -

How to install

1. Put ron-banking in your resources folder
2. ensure ron-banking in your server config folder
3. Enjoy!

Controls

Click E to open GUI

Esc to Exit
